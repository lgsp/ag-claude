Point clé
Le programme couvre toute la panoplie classique de l'algorithmique des graphes. Les deux représentations (matrice vs listes) sont exigibles, ainsi que leurs compromis. Les complexités sont à maîtriser.

4.5.2 — Parcours BFS et DFS

BFS — Parcours en largeur
File FIFO. Visite par niveaux croissants. Distance minimale (non pondérée) = niveau du sommet. Complexité O(V+E).
Invariant : tous les sommets à distance k sont découverts avant ceux à distance k+1.
DFS — Parcours en profondeur
Pile (ou récursion). Timestamps découverte/fin. Détecte les cycles (arête arrière). Tri topologique (ordre décroissant de fin sur DAG). Complexité O(V+E).
Invariant : l'intervalle [d[v], f[v]] est inclus dans [d[u], f[u]] si v est un descendant de u.

Dijkstra — invariant clé

Invariant de Dijkstra
Quand un sommet u est extrait du tas, dist[u] est la distance définitive minimale depuis la source.
Précondition absolue : tous les poids doivent être strictement positifs (ou ≥ 0). Un poids négatif invalide l'algorithme.

Bellman-Ford — principe

Bellman-Ford
Effectuer V−1 passes sur toutes les arêtes. À chaque passe, relâcher chaque arête.
Après V−1 passes, toutes les distances sont optimales (si pas de cycle négatif).
Détection de cycle négatif : si une relaxation est encore possible à la V-ième passe → cycle absorbant.

Floyd-Warshall — récurrence

Récurrence DP
d[i][j][k] = min(d[i][j][k-1], d[i][k][k-1] + d[k][j][k-1])
d[i][j][k] = distance minimale de i à j en utilisant uniquement les sommets {1,…,k} comme intermédiaires.
Initialisation : d[i][j][0] = poids(i,j) si l'arête existe, +∞ sinon, 0 si i=j.

4.5.4 — Arbres couvrants minimaux

AlgorithmeStratégieStructureComplexité
KruskalTri des arêtes par poids, ajout si pas de cycle (Union-Find)Union-FindO(E log E)
PrimCroissance depuis un sommet, ajouter l'arête de poids min vers l'extérieurTas minO((V+E) log V)
Théorème de la coupe (Cut property)
Pour tout coupe (S, V\S) du graphe, l'arête de poids minimal traversant la coupe appartient à tout ACM.
Ce théorème est la base de la preuve de correction de Kruskal et Prim.
